# **App Name**: Kyriba Frontend Replica

## Core Features:

- Menu Map Display: Replicate the main menu map display, including all nodes, icons, and interconnections shown in the original Kyriba portal.
- Interactive Nodes: Implement interactive nodes for each section of the menu map. Clicking a node navigates the user to a corresponding placeholder page.
- Search Functionality: Add a search bar with a basic filter to easily locate main key features shown on the menu map.
- Bank Statements dropdown: Implement Bank Statements dropdown with available search to quickly access main operations.
- "Ready for a Live Demo?" modal popup: Integrate bottom right bar button functionality as found on main UI.

## Style Guidelines:

- Primary color: Subtle light grey (#F0F2F5) to create a clean and professional look.
- Background color: Very light grayish-blue (#E8E9EB) to add depth without being distracting.
- Accent color: Light blue (#3498DB) for interactive elements and highlights, providing a modern and user-friendly feel.
- Body and headline font: 'Inter' sans-serif for a clean, readable, and modern user interface.
- Use flat design icons, primarily in light blue (#3498DB), to match the style of the Kyriba portal.
- Replicate the grid-based layout with consistent spacing and padding for a structured and professional appearance.
- Subtle transitions and hover effects on interactive elements to enhance user experience.
